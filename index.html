🎓 Abhyasa SSC

Your personal study desk for SSC CGL preparation — PDFs, HTML notes, and videos, all in one place. Select a subject from the left to begin.

How to use
  1. Pick a subject from the left sidebar — Reasoning, Maths, GA, English, or Computer.
  2. Choose your local folder for that subject when prompted. The app scans all subfolders automatically.
  3. Click any file to open it — PDFs and HTML files open inline; videos play with progress saved.
  4. Mark files done with the ✓ button. Your progress is saved locally and shown on the Your Progress page.
  5. Upload your Excel schedule (.xlsx) from the Your Progress page to see today's topics each day.
Recent updates
HTML file support Interactive HTML study files now open in a sandboxed viewer alongside PDFs and videos. Filter by type using the HTML tab in the file sidebar.
Subjects panel The left sidebar now shows full subject names. Your Progress and Abhyasa SSC are pinned at the top and bottom for quick access.
Bottom navigation bar Prev and Next controls moved to the bottom bar, with the subject name and current file shown in the centre.
File limit guard Folders with more than 500 files show a warning instead of freezing the browser.
Data migration Existing planner data from the previous version is automatically carried over on first load.
Browser requirement

This app uses the File System Access API, which requires Chrome or Edge (desktop). Firefox and Safari are not supported. Folder access is re-requested each session — this is a browser security requirement, not a bug.
